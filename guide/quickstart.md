# Quickstart

Nexent Agent Factory is a SaaS workspace for OpenAI Codex agent workflow platform.

## Fast Path

1. Open Nexent Agent Factory and choose a bounded pilot workflow.
2. Enter public-safe examples for OpenAI Codex agent workflow platform.
3. Review the generated queue, evidence, or approval state with the owner.
4. Use pricing and checkout links when the pilot needs a paid workspace.
5. Export the resulting report or evidence for the team record.
